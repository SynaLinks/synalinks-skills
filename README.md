# [WIP] Synalinks CLI

A Domain-Specific Coding Assistant for Synalinks

## Features

- **Language & Embedding Model Agnostic**: Bring Your Own Key (BYOK) and use the provider of your choice among Anthropic, OpenAI etc.
- **Domain Specific**: Specifically tailored for Synalinks for better accuracy and relevance
- **Graph-First**: Use a knowledge graph to structure its memory
- **Template based**: Use templates to ensure a correct use of Synalinks library
- **Separation Of Concerns**: Use specific workflows/agents for writing specifications, data modelling and implementation.
- **Continuous Learning**: Learns from its own success and failures
